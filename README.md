## DevTodoNote

## Github repository별 Todo List를 관리 할 수 있는 앱 입니다.

### 기술스택

* Kotlin
* MVVM
* Clean Architecture
* Dagger Hilt
* Jetpack Navigation component
* Room Database
* Lottie
* Retrofit
